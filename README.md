# multi-object-adas-detection

AI-based ADAS perception system using YOLOv8 and OpenCV to detect and track vehicles, pedestrians, bicycles, motorcycles, buses, trucks and traffic lights from road videos.

## Features

- YOLOv8 object detection
- Vehicle, pedestrian and traffic light detection
- Object tracking with unique IDs
- Real-time dashboard with object counters
- FPS display
- Video processing with OpenCV

## Technologies

- Python
- YOLOv8
- Ultralytics
- OpenCV
- NumPy
- Deep Learning
- ADAS Perception

pip install -r requirements.txt
